Что здесь новое и важное

Вкладки "Логистика" и "Таможня" — новые, с NEW badge. Доступны для admin + head_of_logistics / head_of_customs. Backend (services/route_logistics_assignment_service.py) уже существует — это UI заполняет dark spot.

Stats strip — 4 метрики: логистов в команде, паттернов, покрытие (get_uncovered_routes() из существующего сервиса), неназначенные. Warning-стиль на "Неназначено" если больше 0.

Unassigned-alert — раскрытый список заявок без auto-match с reason pill'ами ("Нет pattern'а на Индия", "pickup_country пустой"). Inline кнопка "Назначить логиста" — dropdown выбора пользователя.

Patterns table — сортировка по specificity (exact first — соответствует _match_route_python()). Chip'ы с флагами для origin/destination, wildcard отображается как dashed "* Любой". Статистика использования ("4 за месяц") сразу рядом с правилом — head видит эффективность.

Side panel "Новый маршрут" — живой preview собираемого pattern'а внизу. "Покроет: 1 неназначенную" — сразу показывает impact. Клик на страну "Индия" в coverage-warnings автоматически пре-заполняет это поле.

Coverage warnings справа внизу — страны без pattern'а (derived from get_uncovered_routes() по текущим активным заявкам). One-click "Добавить" → открывает side panel с pre-filled страной.
