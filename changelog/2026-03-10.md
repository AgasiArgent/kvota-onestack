## Новое

- **Уведомления в Telegram** — в меню появился раздел «Уведомления», где можно привязать свой Telegram для получения сообщений о задачах, согласованиях и статусах. При решении обращения автор получает уведомление
- **Чат по КП** — новая вкладка «Комментарии» на странице коммерческого предложения для общения между отделами
- **Журнал обновлений** — страница «Обновления» с индикатором непрочитанного в меню
- **Реестр валютных инвойсов** — сгруппированный реестр с секцией документов в карточке инвойса
- **Маршрутизация закупок** — автоматическое распределение по группам продаж с приоритетом каскада: группа → бренд
- **Автоназначение логистики** — при завершении закупки система автоматически назначает логиста на основе маршрута
- **Журнал звонков** — раскрываемые карточки звонков с деталями встреч и заметками

## Улучшения

- **Поиск клиента при создании КП** — поиск по названию с подсказками, дедупликация по ИНН, отображение контактного лица
- **Фильтрация по менеджеру** — задачи, КП и дашборд показывают только данные текущего менеджера
- **Заметки продаж на странице закупок** — закупщик видит комментарии менеджера
- **Маржа и наценка** — на странице КП отображаются обе величины с формулами расчёта
- **Вес и объём товаров** — данные видны на странице логистики
- **Артикул поставщика** — новое поле в карточке товара
- **Размеры и количество мест** — добавлены в карточку инвойса
- **Обогащение валютных инвойсов** — автоматическое заполнение контрактов, банковских реквизитов и условий оплаты
- **Профиль вынесен из меню** — отдельная ссылка на профиль и кнопка выхода в нижней части панели
- **Уведомление продажам** — менеджер получает Telegram-сообщение при завершении закупки по инвойсу

## Исправления

- Исправлен выбор клиента при создании КП
- Исправлена фильтрация клиентов для менеджеров с несколькими ролями
- Исправлена ошибка дублирования IDN-квоты при генерации
- Исправлены ошибки закупок: загрузка файлов, валидация рабочего процесса
- Исправлено отображение статистики клиентов и валюты позиций
- Исправлена видимость клиентов: доступ руководителя продаж, поиск, валюта
- Исправлена потеря ID сущностей при перегенерации документов
- Исправлена ошибка завершения закупки при пустом POST-запросе
- Исправлено отображение пустого состояния чата после первого сообщения
- Исправлены Telegram-уведомления: URL приложения и запись в журнал
