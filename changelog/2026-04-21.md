## Что это для пользователя

Большой закадровый релиз: закончили перевод OneStack со старого технологического стека на новый. **Никаких новых кнопок в UI не добавлено — но почти всё, что вы делаете, теперь работает быстрее и надёжнее**.

### Что заметите

- **Страницы открываются ровнее** — раньше часть страниц отдавал Python напрямую, часть — Next.js, и переходы между ними были немного дерганые. Теперь весь UI — единый Next.js, переходы мгновенные, почти без «мигания».
- **Сессии больше не слетают между вкладками** — вся авторизация единообразно проходит через Next.js / Supabase. Больше нет сценария «на одной вкладке залогинен, на другой — нет».
- **Ошибки теперь реально видны команде** — раньше Sentry был формально подключён, но из-за пропущенной настройки **за 6 месяцев в него не пришло ни одно событие** (мы нашли и починили это в рамках этого же релиза). Теперь любой необработанный сбой автоматически уходит в Sentry — мы увидим и почистим быстрее.
- **Обновлён Кост-анализ КП** — раньше это была старая FastHTML-страница, теперь это полноценная Next.js-страница с той же логикой расчёта, но в единой визуальной системе остальных страниц.

### Что **не** изменилось

- Логика расчётов (`calculation_engine`) не тронута — вообще ни строчки.
- Базы данных, доступы, роли, workflow — всё на своих местах.
- Адреса страниц (`/quotes`, `/customers`, `/admin/...`) — без изменений.
- Все раcчёты до релиза и после — bit-identical.

## Технические детали

Phase 6C был финальным этапом strangler-fig миграции с FastHTML на FastAPI + Next.js. Конкретно:

### Что убрали

- **FastHTML shell retired** — 25,621 LOC in `main.py` ужались до удаления: все `@rt` handlers перенесены в `legacy-fasthtml/` (архив, никогда не исполняется) + переписаны на Next.js/FastAPI.
- **Docker CMD перешёл на uvicorn** — `python main.py` (FastHTML `serve()`) заменён на `uvicorn api.app:api_app --host 0.0.0.0 --port 5001`. Python-контейнер теперь — чистый FastAPI service.
- **python-fasthtml + fastcore удалены** из `requirements.txt`. Транзитивные `itsdangerous` и `python-multipart` допинены явно (они нужны напрямую).
- **`main.py` удалён целиком** — 7,176 LOC → 0. 8 тестов, которые его импортировали, перетаргечены на `api.app`.
- **~5,600 LOC мёртвых тестов** — тестов, парсящих `main.py` как строку и валидирующих архивированные FastHTML-обработчики.

### Что добавили / починили

- **Новый `services/calculation_helpers.py`** — `build_calculation_inputs`, `resolve_vat_zone`, `_calc_combined_duty`, + константы `COUNTRY_NAME_MAP`, `EU_COUNTRY_VAT_RATES`, `DIRECT_COUNTRY_ZONES`. Вынесены из `main.py` без логических изменений.
- **Middleware relocation** — `SessionMiddleware`, `ApiAuthMiddleware`, Sentry init переехали из `main.py` в `api/app.py`. Outer app (`api_app`) владеет middleware + mount'ит inner sub-app (`api_sub_app`) на `/api`, что сохраняет `/api/*` URL-контракт с Next.js / Caddy / docker-compose healthcheck / JWT guard.
- **Docker HEALTHCHECK** теперь бьёт `/api/health` вместо `/login` (последнее возвращало 404 после архивирования auth shell — контейнер формально был unhealthy, но compose-override спасал).
- **`.dockerignore`** исключает `legacy-fasthtml/` (3.7MB архива) и `frontend/` (отдельный build context) из Python-образа.
- **SENTRY_DSN теперь forward'ится в контейнер** через `docker-compose.prod.yml` (`SENTRY_DSN=${SENTRY_DSN:-}`, `SENTRY_ENVIRONMENT=${SENTRY_ENVIRONMENT:-production}`). Баг предшествовал Phase 6C — `.env` на VPS содержал DSN, но compose не прокидывал его. Сам SDK инициализировался, но `sentry_sdk.Hub.current.client` был `None` всё это время. Теперь `enabled: True`, `dsn_host: o4510363675197440.ingest.us.sentry.io`, test-event отправлен.

### Cost-analysis: port на Next.js + FastAPI

- Новый FastAPI endpoint `GET /api/quotes/{id}/cost-analysis` в `api/cost_analysis.py` + router.
- Новая FSD feature в `frontend/src/features/cost-analysis/` — 4 компонента (`CostAnalysisView`, `SummaryCards`, `WaterfallTable`, `NotCalculated`) + форматтеры.
- Старый FastHTML-обработчик `/quotes/{id}/cost-analysis` (306 LOC) архивирован в `legacy-fasthtml/cost_analysis.py`. `/cost-analysis-json` stub удалён.
- 17 pytest тестов на новый endpoint.

### Риски / как откатить

- Весь `@rt` код сохранён в `legacy-fasthtml/*.py` (`# flake8: noqa`, никогда не импортируется в runtime). Если потребуется археология поведения до миграции — git blame + папка на диске.
- Docker rollback: `gh pr revert` на любой PR или `ssh beget-kvota "cd /root/onestack && git revert HEAD --no-edit && git push"`. Деплой откатится за <3 минуты.
- `docker-compose.prod.yml` healthcheck — страховка: если `uvicorn api.app:api_app` не поднимется, deploy workflow фейлится, старый контейнер остаётся живым.
